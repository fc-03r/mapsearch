# 資材マップ検索システム

## 概要
生産管理システムに関する業務の中で、部品の保管場所を確認する作業に時間を要していたため、部品番号から保管場所を検索できるWebアプリケーションを作成しました。
検索した部品の保管場所を資材マップ上で視覚的に確認できるようにし、在庫調査や現場での部品探索を効率化することを目的としています。

## 開発背景
現場在庫とシステム在庫の差異調査を担当する中で、部品の保管場所を都度確認する作業に手間がかかることを課題として感じていました。
そこで、職業訓練校で学んだWeb開発の知識を活かし、業務効率化を目的として本システムを制作しました。

## 主な機能
- 部品番号検索
- 保管場所表示
- 資材マップ上で該当箇所をハイライト表示
- 検索結果の視覚化

## 使用技術
- HTML
- CSS
- JavaScript
- Fetch API

## 工夫した点
- 保管場所を視覚的に確認できるマップ表示
- 検索結果を強調表示することで視認性を向上
